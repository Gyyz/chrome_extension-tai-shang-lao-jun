# -chrome_extension-tai-shang-lao-jun
# 太上老君说常清静经

## 插件简介

专门展示道教经典《太上老君说常清静经》

## 功能特点

### 
- **阅读模式切换**：提供专注阅读模式，减少视觉干扰
- **字体大小调节**：支持12px-24px字体大小调节
- **键盘快捷键支持**：
  - `R` 键：切换阅读模式
  - `Ctrl/Cmd + +`：增大字体
  - `Ctrl/Cmd + -`：减小字体
  - `Ctrl/Cmd + 0`：重置字体大小
- **设置记忆**：自动保存用户的阅读偏好

## 安装方法

### 方法一：开发者模式安装（推荐）

1. **下载插件文件**
   - 下载整个文件夹到本地

2. **打开Chrome扩展管理页面**
   - 在Chrome浏览器地址栏输入：`chrome://extensions/`
   - 或者点击菜单 → 更多工具 → 扩展程序

3. **启用开发者模式**
   - 在扩展程序页面右上角打开"开发者模式"开关

4. **加载插件**
   - 点击"加载已解压的扩展程序"按钮
   - 选择 `chrome_extension-tai-shang-lao-jun` 文件夹
   - 点击"选择文件夹"

5. **完成安装**
   - 插件图标将出现在浏览器工具栏中
   - 点击图标即可打开经文阅读界面

### 方法二：打包安装

1. 在扩展程序页面点击"打包扩展程序"
2. 选择 `qingjingjing-extension` 文件夹
3. 生成 `.crx` 文件后拖拽到Chrome中安装

## 使用说明

### 基本操作
- **打开插件**：点击浏览器工具栏中的插件图标
- **阅读经文**：在弹出的窗口中阅读完整的《常清静经》
- **调整字体**：使用"放大"和"缩小"按钮调整字体大小
- **切换模式**：点击"阅读模式"按钮进入专注阅读模式

### 快捷键操作
- **R**：快速切换阅读模式
- **Ctrl/Cmd + +**：增大字体（适合视力较好的用户）
- **Ctrl/Cmd + -**：减小字体（适合需要小字体的用户）
- **Ctrl/Cmd + 0**：重置为默认字体大小

### 个性化设置
插件会自动记住您的设置：
- 字体大小偏好
- 阅读模式状态
- 下次打开时自动恢复
